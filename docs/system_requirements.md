# System Requirements

## Web Application

FlowDesk is a web-based platform accessible via any modern browser. No installation required for the web version.

### Supported Browsers

| Browser | Minimum Version | Recommended |
|---------|----------------|-------------|
| Google Chrome | 100+ | Latest |
| Mozilla Firefox | 100+ | Latest |
| Microsoft Edge | 100+ | Latest |
| Safari | 16+ | Latest |
| Opera | 90+ | Latest |

**Not supported**: Internet Explorer (any version), older mobile browsers

### Network Requirements

- **Minimum bandwidth**: 1 Mbps download / 512 Kbps upload
- **Recommended bandwidth**: 5 Mbps download / 2 Mbps upload
- **Latency**: < 200ms to api.flowdesk.com for optimal real-time features
- **Ports**: HTTPS (443) must be open — no other ports required
- **Firewall allowlist**: `*.flowdesk.com`, `*.googleapis.com`, `*.gstatic.com`

## Desktop Application

### Windows
- **OS**: Windows 10 (21H2) or later, Windows 11
- **Processor**: 64-bit, 2 GHz or faster
- **RAM**: 8 GB minimum, 16 GB recommended
- **Storage**: 2 GB free disk space
- **.NET Framework**: 4.7.2 or later

### macOS
- **OS**: macOS 12 (Monterey) or later
- **Processor**: Intel or Apple Silicon (M1/M2/M3)
- **RAM**: 8 GB minimum, 16 GB recommended
- **Storage**: 2 GB free disk space

### Linux
- **Distributions**: Ubuntu 22.04+, Fedora 38+, Debian 12+
- **Processor**: 64-bit, 2 GHz or faster
- **RAM**: 8 GB minimum
- **Storage**: 2 GB free disk space
- **Dependencies**: glibc 2.31+, GTK 3.24+

## Mobile Application

### iOS
- **Devices**: iPhone 8 and later
- **OS**: iOS 16 or newer
- **Storage**: 150 MB for app installation

### Android
- **Devices**: Devices with 4 GB RAM or more
- **OS**: Android 12 or newer
- **Storage**: 100 MB for app installation

## Hardware Integrations

### VoIP / Phone Channel
- Compatible with SIP-based phone systems
- Supported headsets: Jabra, Plantronics/Poly, Logitech — USB or Bluetooth
- Minimum audio quality: 16 kHz sampling rate

### Screen Sharing
- Requires screen recording permissions (macOS: Security & Privacy → Screen Recording)
- Works with up to 2 monitors
- Maximum resolution: 4K (3840×2160) at 30 fps
