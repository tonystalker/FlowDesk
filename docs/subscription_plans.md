## Upgrading Your Plan

1. Go to **Settings** → **Billing** → **Current Plan**
2. Click **Upgrade**
3. Select the desired plan and confirm
4. Prorated charges apply — you only pay the difference for the remainder of the current billing cycle
5. New features are available immediately after upgrade

## Downgrading Your Plan

1. Go to **Settings** → **Billing** → **Current Plan**
2. Click **Change Plan** → select the lower tier
3. Downgrade takes effect at the end of the current billing period
4. You will retain access to current-tier features until the period ends
5. Data beyond the new plan's limits (e.g., extra knowledge base articles) will be archived, not deleted — upgrade again to restore access
